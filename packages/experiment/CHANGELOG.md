# @modular-prompt/experiment

## 0.1.0

### Minor Changes

- 初回リリース: プロンプトモジュールの比較・評価フレームワーク

  **主な機能**:
  - 複数のプロンプトモジュールを同一条件で比較実行
  - コードベースおよびAI評価の両方をサポート
  - YAML/TypeScript設定ファイル対応（jitiで自動トランスパイル）
  - 統計分析とレポート生成
  - ドライバーキャッシング機能

  **CLI**:
  - `modular-experiment <config>` コマンド
  - テストケース、モジュール、評価器の指定オプション
  - 繰り返し実行による統計分析
